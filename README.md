# Valora — Storytelling Financeiro

Apresentação web interativa com 6 páginas, criada com React, TypeScript, Vite, Framer Motion, ECharts, Lucide React e SheetJS.

## Executar

```bash
npm install
npm run dev
```

Também é possível executar `INSTALAR_E_INICIAR.bat` no Windows.

## Requisitos

- Node.js 20.19+ ou 22.12+
- npm 10+

## Base de dados

A base fictícia está em `public/data/base_ficticia_valora_storytelling.xlsx` e pode ser baixada pelo botão no canto superior direito da apresentação.

## Navegação

- Setas esquerda/direita
- Page Up/Page Down
- Espaço para avançar
- Home/End
- Botões de navegação no topo
